# # Pemesanan Tiket Pesawat
# Aplikasi ini adalah sistem sederhana untuk memesan tiket pesawat berbasis Java menggunakan GUI (Swing). Aplikasi memungkinkan pengguna untuk menambahkan, memperbarui, menghapus, dan memesan tiket pesawat dengan mudah.
#
# # Fitur Utama
# 1. Tampilkan Data Tiket   
# 2. Tambah Tiket Baru 
# 3. Update Tiket
# 4. Hapus Tiket
# 5. Pesan Tiket
#
# # Persyaratan Sistem
# - Java Development Kit (JDK) versi 8 atau lebih baru.
# - IDE (Integrated Development Environment) seperti IntelliJ IDEA atau yang lainnya untuk menjalankan dan mengembangkan aplikasi ini.

# # Cara Menjalankan Aplikasi
# 1. Clone atau Unduh Proyek
# 2. Buka di IDE
# 3. Kompilasi dan Jalankan
# 4. Interaksi dengan Aplikasi
#
# # Cara Menggunakan Fitur
# 1. Tambah Tiket
#    - Klik tombol Tambah Tiket.
#    - Isi nama maskapai, tujuan, dan harga tiket.
#    - Klik OK untuk menyimpan data tiket baru.
#
# 2. Update Tiket
#    - Pilih baris tiket yang ingin diupdate pada tabel.
#    - Klik tombol Update Tiket.
#    - Edit data tiket yang diinginkan.
#    - Klik OK untuk menyimpan perubahan.
# 
# 3. Hapus Tiket
#    - Pilih baris tiket yang ingin dihapus pada tabel.
#    - Klik tombol Hapus Tiket.
#    - Konfirmasi penghapusan.
#
# 4. Pesan Tiket
#    - Pilih baris tiket yang ingin dipesan pada tabel.
#    - Klik tombol Pesan Tiket.
#    - Isi informasi nama, jumlah tiket, tanggal penerbangan, dan waktu penerbangan.
#    - Klik OK untuk menyelesaikan pemesanan.
#
# # Catatan Penting
#   - Pastikan Anda mengisi data seperti harga tiket, jumlah tiket, tanggal, dan waktu penerbangan dengan format yang benar.
#   - Gunakan format tanggal 'yyyy-MM-dd' dan format waktu 'HH:mm' untuk menghindari kesalahan input.
#
